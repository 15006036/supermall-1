### 3.修改配置，新增别名【注】在 DOM 中使用别名，需要加~

- 新增 vue.config.js 文件，module.exports = { configureWebpack: { ... } }

```
  module.exports = {
    configureWebpack: {
      resolve: {
        alias: {
          assets: "@/assets",
          common: "@/common",
          components: "@/components",
          network: "@/network",
          views: "@views"
          /* 由于router和store只在main.js里引用，且其他地方只需要 this.$router就可以，所以不需要 */
        }
      }
    }
  };
```
